## Goal
The goal of this assignment is to practice using the **`print()`** function in Python.

## Instructions
Complete the `printpractice.py` file to output a fake address on three separate lines:

1.  **Line 1:** Your Name
2.  **Line 2:** Address (Do NOT use your real address)
3.  **Line 3:** City, State Zip

### Expected Output Example:
```text
Joe Schmo 
1234 Circle Way 
Pittsburgh, PA, 15228
```

## How to Run Your Program in Codespaces

1.  Open the **Terminal** tab at the bottom of the screen.
2.  Type the command below and press Enter:
    ```bash
    python printpractice.py
    ```
3.  Verify that the output in the terminal matches the expected format.
